How to get started with HTML and CSS


What are HTML and CSS

Hypertext markup language (HTML) and Cascading Style Sheets (CSS) are the basis for all websites. What exactly are HTML and CSS? HTML is used to create the foundation of every single website ever made. It describes how a document should be displayed in a browser using tags, elements, attributes, and more. When the structure of your website is built it needs to look appealing to the website users. Now CSS, a styling sheet language is used to describe how the HTML document appears on a web browser. To make your website look stylish, you need to use CSS rules, selectors, and values.

HTML Attributes: are put inside the opening tag to add additional information to an element to also give it characteristics. Attributes always have a name and value to them.



HTML Inline and Block Elements: There are two default values that every element can either have inline, or block.


Inline elements: only take up space that they need and stay on their preexisting line.


Block elements: start on a new line and are given extra spacing taking up the entire line.



CSS Basics


CSS Rules: groups one or more HTML elements so they can be styled.


Selectors: are the HTML element that the styles that will be applied.


Property: a specific style type that is added to the selector.
